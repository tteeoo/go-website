Used by me

  • Operating system: Artix Linux
    • A derivative GNU/Linux distribution of Arch Linux but without systemd. I don't dislike systemd or anything, I just decided to try something different. I might switch back to Debian soon though.

  • Text editor: Neovim
    • The next evolution in the standard text editor lineage (ed -> ex -> vi -> vim -> nvim).

  • Graphical environment: dwm under Xorg

sest

Short for "secure strings", sest is a local, command-line password manager. It uses encryption and cryptographic hashes to securely store your passwords (or really any string of characters) on your own device. sest has only been tested on GNU/Linux, but it should work on any UNIX-like OS. sest is written in Go.
