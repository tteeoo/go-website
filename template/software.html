sest

Short for "secure strings", sest is a local, command-line based password manager. It uses encryption and cryptographic hashes to securely store your passwords (or really any "string" of characters) on your own device. You interact with it using the terminal (perfectly usable in my opinion), although a more user friendly interface could be added to it. sest has only been tested on Linux, but it should work on any UNIX-like OS.
